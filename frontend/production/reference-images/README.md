# Reference Images for Design

This folder contains reference images to help design the booking system to match The Scenic Inn's aesthetic.

## How to Use

1. **Add your images here** - Place restaurant photos, interior shots, logo files, menu designs, or any visual references
2. **Supported formats**: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
3. **Naming suggestions**:
   - `interior-1.jpg`, `interior-2.jpg` - Restaurant interior photos
   - `exterior.jpg` - Building exterior
   - `food-1.jpg`, `food-2.jpg` - Food photography
   - `logo.png` - Logo files
   - `menu-design.jpg` - Existing menu designs
   - `color-scheme.jpg` - Color palette references

## What to Include

- Restaurant interior/exterior photos
- Food photography
- Existing branding materials
- Color scheme references
- Typography examples (if any)
- Menu designs
- Any other visual elements that represent the restaurant's style

## Analysis

Once images are added, I can analyze them to:
- Extract color palettes
- Understand the aesthetic style
- Match typography choices
- Create cohesive design elements
- Ensure the booking system feels integrated with the restaurant's brand
